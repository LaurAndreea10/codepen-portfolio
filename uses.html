Uses

Stack-ul meu de zi cu zi

Inspirat de uses.tech. Lista celor mai folosite tools, software și config-uri în proiectele mele front-end. Actualizat la fiecare schimbare reală, nu la modă.

💻 Editor & Terminal

  • VS CodeEditor principal. Tema: One Dark Pro. Font: Fira Code cu ligatures.
  • Extensii esențialeES7+ React snippets, Prettier, ESLint, GitLens, Live Server, Tailwind IntelliSense, Error Lens, Path Intellisense, Auto Rename Tag.
  • TerminalWindows Terminal cu PowerShell 7. Oh My Posh pentru prompt customizat.

🛠 Stack tehnic

  • Front-end frameworkReact 18 cu Vite 5. TypeScript pentru proiecte cu 3+ componente.
  • StylingCSS variables + Tailwind pentru proiecte noi. CSS modules pentru cele moștenite.
  • ChartsRecharts pentru proiecte React (declarativ, TS-friendly). Chart.js pentru HTML pure.
  • Build & deployVite + GitHub Actions → GitHub Pages. CI/CD configurat de la primul commit.
  • Bundlingesbuild via Vite pentru dev. Rollup pentru production. Bundle target ~50KB pentru proiecte simple, ~300KB pentru SaaS.

🎨 Design & UX

  • WireframingExcalidraw pentru schițe rapide. Fără culori, doar ierarhie.
  • DesignFigma (free tier) pentru mockup-uri detaliate când e nevoie.
  • CuloriCoolors pentru palete. Realtime Colors pentru testare contrast.
  • IconițeLucide (open source, consistent). Heroicons pentru proiecte cu Tailwind.
  • FonturiInter pentru UI. JetBrains Mono pentru cod. Fluid typography cu `clamp()`.

🌐 Browser & testing

  • Browser principalChrome cu profil dedicat dev. Firefox pentru testare cross-browser.
  • DevTools extensiiReact DevTools, axe DevTools (a11y), Lighthouse, Wave, ColorZilla.
  • PerformanceLighthouse CI pe GitHub Actions. Web Vitals extension pentru monitoring local.

🚀 Hosting & servicii

  • Static hostingGitHub Pages pentru proiecte personale. Gratuit, deploy automat, custom domain support.
  • Cu backendVercel pentru proiecte cu API routes (free tier).
  • StoragelocalStorage pentru proiecte personale. Supabase pentru proiecte care au nevoie de DB real.
  • CDNcdnjs.cloudflare.com pentru biblioteci externe în CodePen pens.

📚 Learning & reference

  • DocsMDN pentru standards. React docs (cele noi). web.dev pentru performance + a11y.
  • CursesfreeCodeCamp, LinkedIn Learning, Anthropic Academy.
  • YouTubeTheo, Web Dev Simplified, Fireship pentru news rapide.
  • AIClaude pentru pair programming, debugging, code review. Folosesc API-ul în proiecte AI-native.

📝 Productivity

  • Roadmap proiecteNotion pentru planning, ideas backlog, decision log.
  • Time trackingToggl pentru a vedea unde se duc orele.
  • FocusSpotify (instrumental playlists). Pomodoro technique pentru sesiuni lungi.

Lista se schimbă rar. Adaug ceva nou doar dacă l-am folosit consistent 2+ săptămâni și înlocuiește ceva mai puțin bun.
