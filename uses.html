Uses · Stack de zi cu zi

Tot ce folosesc pentru a construi proiecte front-end

Inspirat de uses.tech. Lista celor mai folosite tools, software și config-uri. Actualizat la fiecare schimbare reală, nu la modă — adaug ceva nou doar dacă l-am folosit consistent 2+ săptămâni și înlocuiește ceva mai puțin bun.

💻 Editor & Terminal
🖊️
VS Code
Editor principal. Temă: One Dark Pro. Font: Fira Code cu ligatures activate. Settings sync pe GitHub pentru backup automat.
Editor principal
⬛
Terminal
Windows Terminal cu PowerShell 7. Oh My Posh pentru prompt customizat cu branch Git, timp de execuție și status cod de ieșire.
Daily driver
Extensii VS Code esențiale
ES7+ React Snippets Prettier ESLint GitLens Live Server Tailwind IntelliSense Error Lens Path Intellisense Auto Rename Tag Thunder Client Peacock
🛠 Stack tehnic
⚛️
React 18 + Vite 5
Framework principal pentru proiecte cu 3+ componente. Vite pentru dev server ~50ms și HMR instant. create-vite ca punct de start standard.
Stack principal
🔷
TypeScript
Pe orice proiect cu 3+ componente sau 300+ linii. Nu pentru strictness de dragul strictness-ului — pentru refactor safe și autocomplete real în IDE.
Type safety
🌊
Tailwind CSS
Pe proiecte noi. Elimină overhead-ul de naming conventions și îți dă consistență fără să construiești design system propriu. Cu Prettier plugin pentru sortare clase.
Styling preferat
📊
Charts
Recharts pentru proiecte React (declarativ, TS-friendly, compose-able). Chart.js pentru HTML pur. Nu amestec niciodată ambele în același proiect.
Data viz
🔊
Web Audio API
Pentru proiecte cu sunet procedural (BASKET VS AI, Excel-Quest). Prefer native API față de librării — mai mult control, zero dependențe, bundle mai mic.
Nativă, fără librării
🚀
GitHub Actions + Pages
CI/CD de la primul commit pe orice proiect major. Workflow standard: npm ci → npm run build → deploy pe gh-pages branch. Zero costuri.
0€ hosting
De ce acest stack și nu altul: Vite + React + TypeScript + Tailwind acoperă 90% din nevoile unui proiect front-end personal, cu cel mai bun DX disponibil acum. GitHub Actions elimină complet costurile de hosting și CI. Combinația funcționează de la un prototip de o zi la un SaaS modular de 6 module.
🎨 Design & UX
✏️
Excalidraw
Wireframing rapid. Fără culori, fără fonturi — doar ierarhie și structură. Dacă wireframe-ul nu e clar în Excalidraw, nu va fi clar nici în browser.
Wireframing
🎭
Figma (free tier)
Pentru mockup-uri detaliate când proiectul merită. Free tier e suficient pentru proiecte personale — 3 proiecte active simultan e mai mult decât am nevoie.
Mockup-uri
🎨
Culori & contrast
Coolors pentru palete. Realtime Colors pentru testare contrast live. WCAG minimum AA pe toate proiectele publice.
Accesibilitate
🔤
Fonturi
Inter pentru UI (cel mai bun pentru densitate mare de text). JetBrains Mono sau Fira Code pentru cod. Fluid typography cu clamp() peste tot.
Typography
🔣
Iconițe
Lucide pentru proiecte standalone (open source, consistent, tree-shakeable). Heroicons pentru proiecte cu Tailwind. Nu amestec librării de iconițe.
Open source
🌐 Browser & Testing
🔵
Chrome DevTools
Profil dedicat dev. Extensii: React DevTools, axe DevTools (a11y), Lighthouse, Web Vitals, ColorZilla. Firefox pentru cross-browser final check.
Browser principal
⚡
Lighthouse CI
Pe GitHub Actions — rulează automat pe fiecare push. Target: Performance 90+, Accessibility 95+, SEO 90+. Alerte dacă scade sub threshold.
Performance
🚀 Hosting & Servicii
🐙
GitHub Pages
Static hosting pentru toate proiectele personale. Gratuit, deploy automat, custom domain support. 20+ proiecte live pe același cont, 0€/lună.
0€/lună
▲
Vercel
Pentru proiecte care au nevoie de API routes sau environment variables. Free tier acoperă orice proiect personal. Preview deployments pe fiecare PR.
Cu API routes
💾
Storage
localStorage pentru proiecte personale (simplu, 0 setup). Supabase când am nevoie de DB real și auth — free tier suficient. Nu IndexedDB — prea verbos pentru complexitatea mea.
Local-first
🤖
Claude (Anthropic API)
Pair programming, debugging, code review. Integrat direct în proiecte AI-native (Link Video Editor Studio). Certificare Claude Code in Action completată.
AI-native dev
