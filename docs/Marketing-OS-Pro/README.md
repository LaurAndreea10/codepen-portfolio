# 📊 Marketing OS Pro

> The marketing command center for self-taught builders. Four production-grade tools — Campaign Studio, Content OS, Brand Kit Generator, SEO & Funnel Analyzer — in one editorial-grade dashboard.

[Live Demo](https://laurandreea10.github.io/codepen-portfolio/marketing-os-pro.html)

## Project Origin

After shipping CRM, content, and marketing micro-tools, Marketing OS Pro creates the missing operations layer: a single workspace that ties campaigns, content, brand systems, SEO, and funnel analysis together.

It is built as a single-file, zero-backend GitHub Pages project with bilingual RO/EN support and an editorial dark UI.

## Modules

### 01 / Campaign Studio

- UTM Builder with live preview and copy
- A/B Test Significance Calculator using a two-tailed z-test
- Google Ad Editor with live preview and character-limit validation

### 02 / Content OS

- Editorial calendar with color-coded posts per channel
- Mock AI copy generator with channel and tone controls
- Readability + Hashtag Lab

### 03 / Brand Kit

- Palette generator with color harmony options
- Aesthetic directions: editorial, tech, organic, brutalist, luxury
- Typography pairings
- Logo mockups on dark, light, and gradient surfaces

### 04 / SEO & Funnel Analyzer

- SEO audit with weighted scoring
- Animated SVG score circle
- Funnel lab with drop-off detection and optimization tips

## Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Fraunces, Inter Tight, JetBrains Mono
- GitHub Pages

## Notes

The app is intentionally client-side and zero-backend. The AI mode is represented as a prepared UI layer; production API integration should be done through a secure backend proxy, not by exposing API keys in the browser.
