# Conversion Funnel Visualizer

> Micro-tool Marketing-Tech pentru vizualizarea funnel-urilor de conversie, drop-off rates și bottleneck analysis.

[Live Demo](https://laurandreea10.github.io/codepen-portfolio/conversion-funnel-visualizer.html)

## Features

- Funnel customizabil cu etape editabile
- Volume per stage
- Drop-off rate per etapă
- Overall conversion rate
- Bottleneck detection
- Presets: B2B SaaS, e-commerce, lead gen, email
- Channel selector
- Copyable report
- RO/EN toggle
- Dark/Light mode
- High contrast mode
- localStorage state pattern
- Zero backend, GitHub Pages deployable

## Why it exists

Funnel reports devin greu de citit când cifrele sunt separate în spreadsheet-uri. Conversion Funnel Visualizer pune toate etapele pe un singur ecran și scoate automat la suprafață etapa cu pierderea cea mai mare.

## Stack

- HTML
- CSS custom properties
- Vanilla JavaScript
- localStorage
- GitHub Pages
