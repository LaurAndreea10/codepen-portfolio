## [1.3.0] — 2026-05-15

### Added

- Campaign Library — istoricul campaniilor cu filter pe canal, status, perioadă
- Empty states informative pentru toate cele 6 module
- Keyboard shortcuts: K, F, B, C și ?
- Help overlay cu lista completă de shortcuts

### Changed

- KPI thresholds configurabile per utilizator
- Funnel drop-off rate se actualizează în timp real

### Fixed

- Budget pie chart nu mai sare la 0% când totalul depășea 100%
