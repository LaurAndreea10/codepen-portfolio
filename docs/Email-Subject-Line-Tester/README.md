# Email Subject Line Tester

> Micro-tool Marketing-Tech pentru testarea subiectelor de email: lungime, cuvinte de impact, emoji, clarity score și risc de spam.

[Live Demo](https://laurandreea10.github.io/codepen-portfolio/email-subject-line-tester.html)

## Ce face

Email Subject Line Tester oferă un scor rapid pentru un subiect de email înainte să fie trimis într-o campanie. Tool-ul rulează local în browser, fără backend și fără tracking.

## Features

- Scor general 0–100
- Analiză lungime: caractere și cuvinte
- Detectare emoji și recomandare de limitare
- Cuvinte de impact: gratuit, checklist, audit, rapid, today/free etc.
- Cuvinte cu risc de spam: garantat, 100%, click aici, limited offer etc.
- Clarity score
- Preview tip inbox
- Raport copyable pentru brief sau meeting
- Bilingv RO/EN
- Dark/Light theme
- GitHub Pages ready

## Stack

- HTML
- CSS custom properties
- Vanilla JavaScript
- localStorage pentru limbă și temă
- Zero backend
