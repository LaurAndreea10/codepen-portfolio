# Email Subject Line Tester · Changelog

All notable changes to this Marketing-Tech micro-tool are documented here.

## [1.0.0] — 2026-05-19

### Added

- Initial release: subject line scoring 0–100
- Length analysis by characters and words
- Emoji detection
- Impact words detection
- Spam-risk words detection
- Clarity score
- Inbox preview
- Copyable report
- RO/EN toggle
- Dark/Light theme
- GitHub Pages deployment
