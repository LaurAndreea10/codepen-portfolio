# Prototip cinematic — Laura Andreea

## Ce am implementat

- Am adăugat fișierul `intro-cinematic-standalone.html` din upload.
- Am creat `index.html` ca hub principal.
- Intro-ul cinematic pornește automat într-un overlay full-screen.
- Există buton `Skip intro`.
- După ~13.5 secunde, utilizatorul ajunge în hub-ul prototipului.
- Am păstrat toate fișierele originale din arhivă.

## Deploy pe GitHub Pages

1. Descarcă arhiva actualizată.
2. Dezarhivează.
3. Urcă toate fișierele în repository.
4. Setează GitHub Pages pe branch-ul `main`.
5. Deschide `https://USERNAME.github.io/REPO/`.

Pagina principală este `index.html`.
