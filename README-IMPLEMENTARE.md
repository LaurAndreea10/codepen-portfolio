# Implementare intro cinematic — v3

Am înlocuit `index.html` cu varianta cinematică nouă, fără iframe.

## Ce include

- Intro afișat direct în `index.html`
- Particule pe canvas
- Reveal pe litere pentru `Laura Andreea`
- Tranziție tip cortină
- Buton `Intră în prototip`
- `Replay intro` în hub
- Toate fișierele originale păstrate
- `intro-cinematic-standalone.html` păstrat separat

## GitHub Pages

1. Urcă toate fișierele din arhivă în repo.
2. Asigură-te că `index.html` este în root.
3. Deschide GitHub Pages. Intro-ul trebuie să apară imediat.

## Test rapid

```bash
python3 -m http.server 8000
```

Apoi deschide `http://localhost:8000`.
