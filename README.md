# 🚀 CodePen Portfolio

A modern portfolio website built to showcase interactive CodePen projects with live preview, filters, search, dark/light mode, and bilingual support (RO/EN).

## 🌐 Live Demo

👉 https://laurandreea10.github.io/codepen-portfolio/

---

## ✨ Features

* 🔴 Live preview for each project
* 🔍 Instant search (name, tags, description)
* 🧩 Filters (Game, Utility, UI, Challenge)
* 🌙 Dark / ☀️ Light mode
* 🌍 Romanian / English toggle
* 📱 Fully responsive design
* 🎨 Modern UI with animations
* 📚 Mini case study for each project

---

## 🛠️ Technologies

* HTML5
* CSS3
* JavaScript
* GitHub Pages
* CodePen projects
