About Marketing OS

Marketing OS este workspace-ul meu Marketing-Tech: benchmarks, UTM-uri consistente, ROI, lead magnets și briefs într-un singur fișier HTML. 0 backend, 0 costuri, deploy pe GitHub Pages.

Marketing OS is my Marketing-Tech workspace: benchmarks, consistent UTMs, ROI, lead magnets and briefs in one HTML file. 0 backend, 0 costs, GitHub Pages deploy.

Stack & decisions

  • Single-file HTML — 0 build, 0 backend, 0 dependencies.
  • localStorage pentru templates/campanii.
  • Bilingv RO/EN nativ.
  • Export JSON pentru backup.
